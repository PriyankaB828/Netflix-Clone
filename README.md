# Netflix Clone – Web Application

🎬 A responsive Netflix-inspired web application built using React and Tailwind CSS. It allows users to browse movies, search content, and watch trailers with a modern UI experience.

---

## ✨ Features
- 🔍 Real-time search functionality  
- 🎥 Trailer popup modal (YouTube integration)  
- 🌙 Dark/Light mode toggle  
- 🎨 Responsive and modern UI design  
- ⚡ Smooth hover effects and transitions  

---

## 🛠️ Tech Stack
- React.js  
- Tailwind CSS  
- JavaScript  
- YouTube Embed API
